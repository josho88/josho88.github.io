Training

What we offer

With so many resources available online, it's hard to know where to start when trying to learn skills like R programming, machine learning and data visualisation. We tailor our training to your business needs so you don't waste time searching for the right training course for your organisation.

We offer 1:1 or small group sessions, in-house or online, to both large and small organisations. If you are interested in becoming skilled at R programming, building Shiny web apps or fitting predictive models with h2o.ai, get in touch via our contact page to discuss suitable training options. Our courses are always bespoke, but examples of areas we cover can be found below.

get in touch

Intro to R Programming

Learn about importing data, connecting to databases, assigning objects, functions, summarising data and basic visualisations.

R for Actuaries

Learn about the R packages and functions that actuaries can use to help with pricing, experience analyses and statistical modelling. This course can help you transition your existing models to an R environment.

Reporting with Shiny

Shiny is a fantastic tool that can turn your data analyses into user friendly web apps and interactive dashboards tailored to your business needs.

Deep Learning & Text Analytics

Discover how deep learning technologies like image recognition, machine translation and text generation could be applied in your business.

Machine Learning with H2O

H2O is a leading machine learning platform that allows you to easily build and deploy models from R, Python, Scala, Java or the H2O web UI.

Data Visualisation in R

We teach you how to explore data visually using tools like R shiny, Plotly and ggplot2 so you can create your own bespoke dashboards and engaging reports.
